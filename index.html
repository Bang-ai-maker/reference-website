Naming Convention

Commit Messages

Write clear and concise commit messages that describe the changes made. Use the imperative mood and start with a capital letter. Limit the message to 72 characters or less.

  1. Cut two lemons in half and squeeze the juice into an oven-safe dish. Save a couple of lemon halves. Add about two cups of water.
  2. Put the lemon mixture into the oven and set the oven temp to 250 degrees.
  3. Allow the water to reach a boil, then turn off the oven. Let the steam circulate in the oven for a few minutes until it’s cool enough to work inside, but still warm to the touch.
  4. Remove the lemon water from the oven and place it next to you on the open door. Dunk a scrubby sponge into the lemon water and wipe down the oven interior and racks.

HTML Tags

Headings

The six levels of headings are represented by the h1 to h6 tags. Use h1 for the main heading of the page and use the other headings for subheadings as appropriate.

Paragraphs

Use the p tag to represent paragraphs of text.

Lists

There are three types of lists:

  • Unordered lists (ul): Use for lists without any specific order
  • Ordered lists (ol): Use for numbered lists
  • Definition lists (dl): Use for lists of terms and their definitions

Links and Images

Use the a tag for links and the img tag for images. For images with captions, use the figure and figcaption tags.

Quotes

Use the q tag for short quotations and the blockquote tag for longer quotations. Use the cite tag to indicate the source of the quotation.

Text Formatting

Use the em tag for emphasis, the strong tag for strong emphasis, the b tag for bold text, the i tag for italic text, and the small tag for small text.

Content Structure

Use the header tag for the header of the page, the footer tag for the footer of the page, and the main tag for the main content of the page. Use the section tag to group related content, the article tag for standalone content that can be syndicated, and the nav tag for navigation links.
